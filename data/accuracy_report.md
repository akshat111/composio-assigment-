## 3. Top Five Recurring Mistakes Identified

1.  **Gating Policy Misattribution:** The agent assumed that if a software requires a corporate plan for production operations (e.g. QuickBooks, HubSpot), developer API keys are also paid-only. In reality, both offer free developer sandbox portals.
2.  **API Key Deprecations:** Stating that "API key" is a valid active authentication method for platforms that transitioned entirely to OAuth2 or Token architectures (e.g. HubSpot private app tokens).
3.  **Domain String Truncation:** Scrapers capturing incomplete URL domains (e.g. `https://paygent` instead of `https://paygent.com`) due to text boundary splits in search snippets.
4.  **Fallback Evidence URL Routing:** Defaulting to generic query search directories (`google.com` or corporate homepages) instead of developer subdomains when encountering HTTP 403 blocks on doc servers.
5.  **Templated Description Generation:** Relying on simple string formatting templates (`"Software application in..."`) instead of semantic extraction of the actual business function.

---

## 4. Top Five Key Improvements Recommended

1.  **Dynamic Sandbox Probe:** Configure the search tool to explicitly check if the query `"{app} developer sandbox create free"` returns an active signup page before assigning the `access_tier` verdict.
2.  **Strict URL Domain Validation:** Implement regex filters to discard evidence URLs that do not resolve to subdomains containing `developer.`, `docs.`, `api.`, or `git`.
3.  **Authentication Spec Matrix:** Cross-check the word occurrences of `Basic Auth`, `Bearer`, `OAuth 2.0`, and `x-api-key` against verified document pages, applying boolean checks to avoid mapping generic tokens.
4.  **Inlined Standalone Compilation:** Inject the JSON metrics directly into the frontend code to eliminate server requirements and CORS block vulnerabilities.
5.  **Continuous Composio Sync:** Hook the verification agent directly into the live Composio `/toolkits` endpoint rather than static catalogs to ensure real-time accuracy on MCP server listings.
